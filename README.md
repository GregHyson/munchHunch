# munchHunch
Locate nearby munch spots for you and friends


Avoid the google ecosystem as much as possible, to avoid unnecessary billing.
Use Leaflet on the frontend to display a map with OpenStreetMap tiles
I want to be able to do a radius around us distance wise, then take the overlap, and search that area for restaurants. 
Then I want to check the walking distance for both of us to that place. 
I would like to also be able to see the entries on the map as hyperlinks to the business listing.
